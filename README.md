# 加时代购物APP

> 该项目是基于mui+mint+vue2.x+vue-router+vuex+webpack制作的
Hybrid APP项目，样式使用vue移动端mint ui框架，原生手机能力
偏重于mui框架，目前只实现了部分功能。

## Start

``` bash
npm install
```

## Develop

``` bash
# serve with hot reload at localhost:8080
npm run dev
```

## Build

``` bash
# build for production with minification
npm run build
```
